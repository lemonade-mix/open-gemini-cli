# Contributing

Contributions welcome.

## Setup

```bash
git clone https://github.com/limkcreply/open-gemini-cli
cd open-gemini-cli
npm install
npm run build
```

## Development

```bash
npm run start          # Run the CLI
npm run test           # Run tests
npm run lint           # Check code style
npm run preflight      # Run all checks before PR
```

## Pull Requests

1. Open an issue first to discuss the change
2. Keep PRs small and focused
3. Run `npm run preflight` before submitting
4. Update docs if adding user-facing changes
